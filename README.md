## Running the Project

### Prerequisites

- Node.js
- Rust
- Tauri CLI

### Development

1. Clone the repository:

```sh
git clone https://github.com/Jacob-Walton/DB-Admin.git
```

2. Navigate to the project directory:

```sh
cd DB-Admin
```

3. Install the dependencies:

```sh
npm install
```

4. Start the development server:

```sh
npm run tauri dev
```

### Build the project:

```sh
npm run tauri build
```

### Run the project:

```sh
npm run tauri run
```
